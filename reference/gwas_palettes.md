# List available GWAS palettes

List available GWAS palettes

## Usage

``` r
gwas_palettes()
```

## Value

A character vector of palette names.

## Examples

``` r
gwas_palettes()
#>  [1] "default"       "colorblind"    "vibrant"       "pastel"       
#>  [5] "dark"          "nature"        "science"       "lancet"       
#>  [9] "nejm"          "aaas"          "brewer_set1"   "brewer_set2"  
#> [13] "brewer_paired" "brewer_dark2" 
```
