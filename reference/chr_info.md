# Chromosome information for common species

Built-in chromosome lengths and centromere positions for human, mouse,
and cattle genomes. For other species, use
[`chr_info_ucsc()`](https://bczech.github.io/ggwas/reference/chr_info_ucsc.md)
to fetch data from UCSC or supply a custom data.frame.

## Usage

``` r
chr_info_human(build = "hg38")

chr_info_mouse(build = "mm39")

chr_info_cattle(build = "ARS-UCD1.2")
```

## Arguments

- build:

  Genome build (default: latest).

## Value

A data.frame with columns: `chr` (integer), `length`,
`centromere_start`, `centromere_end`.
